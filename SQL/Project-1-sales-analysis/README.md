# 📊 Sales & Customer Insights Analysis (SQL Project)

## 👨‍💻 Overview
This project analyzes sales and customer data using SQL to extract meaningful business insights.  
It simulates a real-world retail analytics scenario where data is used to support decision-making.


## 🎯 Business Questions Answered
- What is the total revenue generated?
- Which products generate the most revenue?
- Who are the top customers by spending?
- Which customers are repeat buyers?
- How do sales trend over time
  

## 🛠 Tools Used
- PostgreSQL (pgAdmin)
- SQL


## 🗂 Dataset Structure
- customers (customer information)
- orders (order details)
- order_items (products per order)
- products (product details)


## 📊 Key Insights
- A small number of products generate the majority of revenue
- Some customers contribute significantly more revenue than others
- Repeat customers play an important role in overall sales
- Sales fluctuate across different months


## 📁 Project Files
- queries.sql → All SQL queries used for analysis


## 🚀 Skills Demonstrated
- SQL Joins (INNER JOIN)
- Aggregations (SUM, COUNT)
- GROUP BY & HAVING
- Data analysis thinking
- Turning raw data into insights
